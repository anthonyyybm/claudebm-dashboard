Privacy Policy
Effective date: April 30, 2026
Internal Tool

1. Overview

Boosted Movers Analytics is a private internal tool that connects to the TikTok API to retrieve performance data for the Boosted Movers TikTok account. This policy explains what data is accessed, how it is used, and how it is protected.

2. What Data We Access

Through the TikTok API, this application reads the following data from the Boosted Movers account only:

  • Follower count
  • Video view counts and engagement metrics
  • Video list and basic metadata

Only data belonging to the authorized account owner is accessed. No other TikTok accounts or users are affected.

3. How We Use the Data

All data retrieved is used exclusively for internal analytics — specifically, to display social media performance on a private dashboard accessible only to authorized Boosted Movers team members. The data is not processed for any other purpose.

4. Data Storage

Analytics data may be cached locally or in a private database to reduce API calls and enable historical tracking. This data is not accessible to the public or any external party.

5. No Selling or Sharing

We do not sell, share, rent, or disclose any data obtained through this application to third parties. The data is never used for advertising targeting, resale, or any commercial purpose outside of internal reporting.

6. Data Retention

Data is retained only as long as needed for internal performance tracking. It can be deleted upon request or when no longer operationally relevant.

7. Security

Access to the dashboard and underlying data is restricted to authorized team members. API credentials are stored securely and are not exposed publicly.

8. Changes to This Policy

This privacy policy may be updated as the application evolves. The effective date at the top reflects the date of the most recent revision.

9. Contact

For questions about this privacy policy or data practices, contact: anthony@boostedmovers.com
