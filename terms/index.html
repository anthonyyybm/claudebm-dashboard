Internal Tool

1. About This Application

Boosted Movers Analytics is a private, internal dashboard used exclusively by Boosted Movers to monitor social media performance. The application connects to the TikTok API to read account-level data — including video statistics and follower count — for display on a private internal dashboard.

This application is not a public-facing product and is not available for use by outside parties.

2. Data Access

The application accesses only the data belonging to the Boosted Movers TikTok account. No data from other accounts is accessed. Access is read-only — no content is posted, modified, or deleted through this application.

3. Data Use

Data retrieved from TikTok is used solely for internal reporting and analytics. It is displayed on a private dashboard accessible only to authorized Boosted Movers team members.

4. No Third-Party Sharing

No data obtained through this application is sold, rented, or shared with any third party. Data is not used for advertising, resale, or any commercial purpose beyond internal performance tracking.
